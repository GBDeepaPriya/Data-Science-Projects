# 💰 Expense Tracker App using Data Science

## 📌 Project Overview

The **Expense Tracker App using Data Science** is an end-to-end data analytics project designed to simulate real-world financial expense tracking and analysis.

This project generates synthetic expense data, performs data cleaning and analysis, visualizes spending patterns, detects overspending, and presents insights through an **interactive Streamlit dashboard**.

It demonstrates practical skills in:

- Data Analysis
- Financial Analytics
- Data Visualization
- Dashboard Development
- Python Programming

This project is ideal for showcasing skills relevant to:

✔ Data Analyst
✔ Business Analyst
✔ Financial Analyst
✔ Data Science roles

---

# 🚩 Problem Statement

Tracking daily expenses manually is difficult and often leads to:

- Overspending
- Poor budgeting
- Lack of financial visibility
- Inefficient financial planning

Many individuals and businesses need a system that:

- Tracks spending
- Categorizes expenses
- Identifies patterns
- Detects overspending
- Generates actionable insights

---

# 💡 Solution

This project builds a **data-driven expense tracking system** that:

✔ Generates realistic synthetic expense data
✔ Cleans and processes transaction data
✔ Performs category and monthly analysis
✔ Detects overspending
✔ Visualizes spending patterns
✔ Provides financial insights
✔ Displays results using an interactive dashboard

# ▶ How to Run Project

## Install Dependencies
pip install -r requirements.txt

## Run Full Pipeline
python main.py

## Run Interactive Dashboard
streamlit run app/streamlit_app.py
